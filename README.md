# HW3_-2509_- Film Data Analyzer

## Команды
- `./run.sh build_generator` — собрать генератор
- `./run.sh run_generator` — сгенерировать data.csv
- `./run.sh build_reporter` — собрать аналитика  
- `./run.sh run_reporter` — сгенерировать report.html
- `./run.sh clear_data` — очистить данные
- `./run.sh structure` — показать структуру
